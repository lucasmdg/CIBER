# Sample datasets

All datasets in this folder are **synthetic** and may be used in publications,
talks, and demos. None represent real organisations or real customer data.

- `actors.json` — placeholder for curated threat actor profiles (used by the
  Threat Intelligence Center).
- `cve-sample.json` — placeholder for the vulnerability list (loaded via
  Prisma seed in the current build).
- `assets-sample.json` — placeholder for the asset inventory (used by the
  Prisma seed).

The Next.js app does **not** read these files at runtime; they are kept for
transparency and for recruiters who want to inspect the data offline.
