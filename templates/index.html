about sentinel

SENTINEL is a local-first malware analysis tool. All processing is done on your machine. Zero data leaves your computer. No cloud APIs, no telemetry, no external requests.

version
1.0.0
engine
flask + python 3
method
signature + heuristic
privacy
100% offline
privacy guarantee
> no file content stored after analysis - memory only
> no logging of file paths or system data to disk
> zero external network requests from the engine
> virustotal lookup opens in your browser - no api key needed
> delete and kill actions require explicit confirmation
capabilities
> file analysis: hash matching, signature detection, entropy calculation, string scanning, macro detection, script analysis
> system scan: process inspection, startup entries, persistence paths, network connections, recent files
> risk scoring: 0-100 score with per-finding severity levels
> actions: delete files, kill processes, disable startup entries, virustotal lookup
