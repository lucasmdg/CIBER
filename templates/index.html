ℹ About SENTINEL

SENTINEL is a local-first malware analysis tool. It performs file and system analysis entirely on your machine. Zero data leaves your computer.

No cloud APIs, no telemetry, no external requests. Every scan runs locally. Optional VirusTotal hash lookup opens in your browser — no API key needed.

🛡 Privacy Guarantee
  • No file content is stored after analysis — analyzed in memory only
  • No logging of file paths or system data to disk
  • Zero external network requests from the analysis engine
  • VirusTotal lookup opens in your browser — no API key, no data sent by this tool
  • All deletion and process kill actions require explicit confirmation
💡 How It Works
  • File Analysis: Computes hashes, checks signatures, calculates entropy, scans for suspicious strings, detects macros, analyzes scripts
  • System Scan: Inspects running processes, startup entries, persistence paths, network connections, and recently modified files
  • Risk Scoring: Each finding is scored, then aggregated into an overall 0–100 risk score
  • Actionable Results: Every finding has clear next steps — delete, kill, disable, or investigate further
